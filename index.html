Console Design

This is a concept project where I asked myself if I designed an xbox handheld game console what would it be like. This is mostly based on my own taste as what I would like in a handheld design. The design would be simple to hold and carry around. It has a flip up screen and looks more like a handhelp controller. It would features unbelievable gaming and multimeda performance. It has a full HD screen, one joystick and a senstive touchpad similar to a laptop, 8 buttons and two triggers for that intense gaming experience. It comes in two colours: Black with a silver centre and white with a black centre. It features on-line mode which enables the user to play games with friends online or even to browse the internet. It features a rechargeable battery and USB port. I created the artwork for a website and packaging making it look more realistic.
